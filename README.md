# WMN Personalizados

Landing page em Vite + React para WMN Personalizados.

## Rodando localmente

Pré-requisito: Node.js.

1. Instale as dependências:
   `npm install`
2. Inicie o ambiente de desenvolvimento:
   `npm run dev`
3. Gere a versão de produção:
   `npm run build`
